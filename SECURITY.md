# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

**Please report security vulnerabilities to:** joseph@thehouseofdeals.com

**Important:** Do NOT open public issues for security vulnerabilities.

We will:
- Respond within 48 hours
- Provide updates every 72 hours until resolved
- Credit you in the fix (if desired)

Thank you for helping keep PopKit secure!
