# PopKit Public Docs Index

The public repository keeps only stable, user-facing reference docs in this folder.

For full product documentation, use the docs site in `packages/docs`.

## Kept In `docs/`

- `docs/AGENT_ROUTING_GUIDE.md`: agent routing behavior and architecture
- `docs/POWER_MODE_ASYNC.md`: power mode async architecture and usage details
- `docs/MCP_WILDCARD_PERMISSIONS.md`: MCP wildcard permission policy and examples
- `docs/errors/README.md`: canonical error-code reference model
- `docs/ROADMAP.md`: high-level public roadmap

## Moved Out

Internal plans, implementation logs, and research snapshots were removed from this public repo cleanup and should live in private/internal planning systems.
