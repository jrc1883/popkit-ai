## Executive Summary

Investigation reveals PopKit has **3 proven mechanisms** for customizing status messages displayed when spawning batches of agents, with varying levels of maturity and UI placement:

1. **Status Line Widgets** (fully implemented but underutilized)
2. **Notification Hooks** (partially implemented)
3. **Power Mode Coordinator** (deep integration point)

**Key Finding**: While PopKit cannot override Claude Code's core "Spawning Batch N" messages (internal to Claude Code), it can enhance the user experience through the status line widget system.

### Recommendation: **MERGE TO v1.0** (Quick Win)

Phase 1 enhancement (status line widget) provides immediate UX value with minimal effort (1-2 hours). Phases 2-3 can be deferred to v2.0.

### Available Context for Customization

**From Power Mode State**:
- `active_issue` - Issue number being worked
- `current_phase` - Phase name (Discovery, Implementation, etc.)
- `phase_index` / `total_phases` - Progress (3/7)
- `streaming.agents_streaming` - List of active agent names
- `streaming.active_streams` - Count of active agents
- `progress` - Percentage (0.0-1.0)

**From Hook Context**:
- `tool_name` - Current tool (Task, Bash, Read, etc.)
- `tool_input` - Parameters
- `session_id` - Current session

### What Cannot Be Customized

**Claude Code Internals**:
- Core "Spawning Batch N" message (generated by Task() tool)
- Batch numbering (managed by Claude Code)
- Subagent creation events (no hook available)
- Main UI message injection (PopKit limited to stderr/status line)

## Implementation Plan (Phase 1)

### Step 1: Add Batch Status Widget

**File**: `packages/plugin/power-mode/statusline.py`

```python
def widget_batch_status(compact: bool = True) -> str:
    """Display batch number and agent information.

    Compact:  "Batch:2 Agents:4"
    Full:     "Batch 2 | Code Explorer, Security Auditor (4 agents)"
    """
    state = load_power_mode_state()
    streaming = state.get("streaming", {})
    batch_num = state.get("batch_number", 0)
    agent_names = streaming.get("agents_streaming", [])

    if not batch_num or not agent_names:
        return ""

    if compact:
        return f"Batch:{batch_num} Agents:{len(agent_names)}"

    # Full format with agent names
    agents_display = ", ".join(agent_names[:3])  # Show first 3
    if len(agent_names) > 3:
        agents_display += f" +{len(agent_names) - 3}"

    return f"Batch {batch_num} | {agents_display} ({len(agent_names)} agents)"
```
