# PopKit v1.0.0-beta.1 Release Notes

**Release Date**: 2025-12-28
**Status**: Beta Release - Ready for Marketplace Publication
**Epic**: #580 Plugin Modularization Complete

---

## 🎉 What's New

PopKit v1.0.0-beta.1 marks the completion of Epic #580: Plugin Modularization. The monolithic plugin has been transformed into **5 focused, modular plugins** that users can install selectively.

### Modular Plugin Architecture

PopKit is now split into 5 specialized plugins:

| Plugin | Version | Purpose | Commands |
|--------|---------|---------|----------|
| **popkit-core** | 1.0.0-beta.1 | Foundation & Power Mode | 11 |
| **popkit-dev** | 1.0.0-beta.1 | Development workflows | 7 |
| **popkit-ops** | 1.0.0-beta.1 | Operations & quality | 5 |
| **popkit-research** | 1.0.0-beta.1 | Knowledge management | 2 |
| **popkit-suite** | 1.0.0-beta.1 | Complete bundle | All |

**Shared Foundation**: popkit-shared v1.0.0 (70 utility modules)

---

## 📦 Installation

### Complete Suite (Recommended for New Users)

Install all features with one command:

```bash
/plugin install popkit-suite@popkit-claude
# Restart Claude Code
```

### Selective Installation (Advanced Users)

Install only what you need:

```bash
# Foundation (required)
/plugin install popkit-core@popkit-claude

# Development workflows
/plugin install popkit-dev@popkit-claude

# Operations & quality
/plugin install popkit-ops@popkit-claude

# Knowledge management
/plugin install popkit-research@popkit-claude

# Restart Claude Code
```

## 📝 Migration Guide

### From Monolithic Plugin (v0.x)

Users with the old monolithic plugin should:

1. **Uninstall old plugin**:
   ```bash
   /plugin uninstall popkit
   ```

2. **Install new suite**:
   ```bash
   /plugin install popkit-suite@popkit-claude
   ```

3. **Restart Claude Code**

All commands remain the same under `/popkit:` namespace!

### Selective Migration

If you only used specific features:

```bash
# Daily development only
/plugin install popkit-core@popkit-claude
/plugin install popkit-dev@popkit-claude

# Add operations when needed
/plugin install popkit-ops@popkit-claude
```
